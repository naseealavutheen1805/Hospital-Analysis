# Hospital Readmission Analysis Dashboard

## Overview
This project focuses on understanding hospital readmissions using Excel. The dataset was cleaned and analyzed using pivot tables, and insights were visualized through an interactive dashboard. The goal is to identify which patient groups have higher readmission rates and what clinical factors contribute to these readmissions.

## Key Metrics
- **Total Diabetes Medication Patients:** 51,205  
- **Average Hospital Stay:** 4.4 days  
- **Readmission Rate:** 46.20%  
- **Most Common Age Group:** 70–80  

## What I Did
- Cleaned and prepared the hospital administration dataset  
- Answered the case study questions using pivot tables  
- Analyzed readmission patterns by age, gender, race, diabetic medication, and medical specialty  
- Built an interactive Excel dashboard with slicers for age, gender, race, and readmission status  

## Insights
- Readmission rates increase with age, especially **70–80** and **80–90** groups  
- Males and females show similar readmission trends across races  
- Patients with **diabetes medications** have slightly higher readmission rates  
- Several medical specialties (e.g., Internal Medicine) handle the largest number of patients  
- Medication count increases as readmission probability increases  
- Hospital stay duration grows steadily with age  

## Deliverables
- Cleaned Dataset  
- Pivot Table Analysis / Solution Guide  
- Excel Dashboard  
- Project Document  

## Conclusion
This analysis highlights the key drivers behind hospital readmissions and provides insights that can support better patient care, discharge planning, and readmission reduction strategies.
